Python NEAT
Flappy Bird AI Version 5.0

Using Python with the Pygame framework, I wrote a basic version of Flappy Bird. I then implemented the NEAT genetic algorithm. This algorithm randomly create "genomes" (neural networks) for each bird in the population. After each generation, the bird that has the highest fitness value is "mutated" (weights/ connections/ layers are randomly adjusted) to create the next generation of birds. I used 3 input neurons (y position of bird, x distance from top pole, x distance from bottom pole) and 1 output neuron (jump = True/False). The activation function I used to evaluate the output is tanH (if tanH > 0.5: jump = True). After 3 generations with a population size of 10, one bird was able to pass over 18000 poles (I dozed off at that point). This was another amazing application of neural networks!

Python Tensorflow
MattBot Version 12.0

This is a chatbot (Mattbot as I call it) I built using Tensorflow in Python. The deep learning model I implemented makes use of 2 hidden layers (8 neurons each/ n_epoch=1000/ batch_size=8). When the user enters a sentence, that sentence is formatted into a list of 0s and 1s ("one hot encoded"). This is used as input to train the network. My current model doesn't classify all user imputs the way that I intend it to, but this is ONLY version 12.0. I hope I can somehow make some pulls from Reddit and build a MUCH MUCH bigger data set.
